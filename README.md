# Boston Women Entrepreneurship Incubator Tracker

The **Boston Women Entrepreneurship Incubator Tracker** is a comprehensive data management and analytics system designed to support the tracking, analysis, and insights generation for startups, mentors, investors, and events within the Boston women entrepreneurship ecosystem.

---

## Project Overview

This project focused on creating an end-to-end data system to help incubators, mentors, and stakeholders efficiently manage operations and make data-driven decisions. The system integrates both **relational (MySQL)** and **NoSQL (MongoDB)** databases to handle structured and semi-structured data.

---

## Key Features

- **Database Design**
  - Designed **EER/UML diagrams** and implemented a **fully normalized relational database**.
  - Enables management of:
    - 120 startups
    - 80 mentors
    - 60 investors
    - 60 events

- **Automated Data Pipelines**
  - Developed **Python scripts** to automate MySQL queries (10+ simple and advanced queries).
  - Performed basic data analysis and generated visual insights using **Matplotlib** and **Seaborn**.
  - Identified top-performing startups, high-impact mentors, and high-growth industries.

- **MongoDB Aggregation**
  - Implemented **aggregation pipelines across 14 core and junction collections**.
  - Executed multi-condition queries to produce actionable insights on:
    - Funding patterns
    - Investor and mentor engagement
    - Event participation trends

---
